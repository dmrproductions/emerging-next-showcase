### Quick Setup

1. **Create a new GitHub repository:**
   - Go to [github.com/new](https://github.com/new)
   - Name it: `emerging-next-showcase`
   - Make it **Public**
   - Click "Create repository"

2. **Upload files:**
   - Click "uploading an existing file"
   - Drag and drop `index.html` from this folder
   - Commit the file

3. **Enable GitHub Pages:**
   - Go to repository **Settings**
   - Click **Pages** (left sidebar)
   - Under "Source", select **main** branch
   - Click **Save**

4. **Get your live URL:**
   - Wait 1-2 minutes
   - Your site will be live at: `https://YOUR-USERNAME.github.io/emerging-next-showcase/`

### Embed on Weebly

Once deployed, add to your Weebly site using an iframe:

```html
<iframe src="https://YOUR-USERNAME.github.io/emerging-next-showcase/" 
        width="100%" 
        height="1200" 
        style="border:none;">
</iframe>
```
